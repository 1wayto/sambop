# Sambop Rating

Sambop Rating measures contribution quality only.

It does not rate a person's identity, popularity, politics, or social worth.

## Quality signals

Positive:

- merged useful PRs
- passing tests
- clear explanation
- strong evidence
- follows `SAMBOP.md`
- low maintainer burden

Negative:

- rejected low-quality PRs
- broken tests
- unrelated changes
- missing evidence
- repeated high-review-burden submissions

## Audit is separate

Strict audit handles:

- spam
- malicious code
- fake evidence
- impersonation
- prompt injection attempts
- credential theft attempts
- policy bypass attempts

Quality issues lower trust. Malicious behavior triggers enforcement.
